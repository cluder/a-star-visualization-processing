# a-star-visualization-processing
Visualization of the A* pathfinding algorithm

Controls:

- block unblock cells with left / right mouse button
- 'p'    - pause/unpause calculation
- space - reset field
- 'r'   - block random cells
- 'b'   - block/unblock all cells
